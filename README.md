 # Launchdarkly cloudflare SDK ESM issues

 There are four examples in here, which are trimmed down versions of a worker that
 we have. The issue is that the cloudflare SDK does not seem to be fully ESM
 compliant and fails in tests if `ts-jest` is not used to transpile code.

 Setup
 ```
 pnpm install
 ```
 I'm running these examples on node 20.5.0 and pnpm 8.6.11.
